What is the difference between Margin, Border, and Padding?

Basics

Think of elements as rectangles of pixels

Margin

  1. Margin dictates space outside the element
  2. Useful for changing an element's position on the page
  3. Useful for setting distance between elements

Padding

  1. Padding dictates space inside the element
  2. Useful for adding space between content and borders
  3. Userful for changing the size of an element e.g. create a larger button

Border

  1. Boundary between what's defined as inside an element and outside an element
  2. Be default set to 0
  3. Useful for stylistic purposes

Analogy

There is a room. In the room there is a hamster in a box. There is also an axolotl in an aquarium in the same room. The space between the box/aquarium and the wall of the room is the margin. The space between the box and the aquarium is also the margin. To move the box and the aquarium closer together is like changing the margin between two elements. The space between the hamster and the box, or the axolotl and the aquarium is the padding. To increase the size of the box to give the hamster more room, can be likened to increasing the padding of an element. The border is the walls of the box or the walls of the aquarium. Making the glass walls on the aquarium thicker is like increasing the thickness of the border.
